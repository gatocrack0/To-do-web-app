# TODO APP

Aplicación de almacenamiento de tareas por hacer generada con NodeJS y WebPack.

https://gatocrack0.github.io/todojs/

### Notas: 
Reconstruir los módulos de Node
```
npm install
```

Y para construir el build:
```
npm run build
```
